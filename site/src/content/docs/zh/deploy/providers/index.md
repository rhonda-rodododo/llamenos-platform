title: 电话服务提供商

---

Llamenos 通过其 **TelephonyAdapter** 接口支持多个电话服务提供商。您可以随时在管理设置中切换提供商，无需更改任何应用代码。

## 支持的提供商

| 提供商 | 类型 | 计费模式 | WebRTC 支持 | 设置难度 | 最适合 |
|---|---|---|---|---|---|
| **Twilio** | 云端 | 按分钟计费 | 是 | 简单 | 快速入门 |
| **SignalWire** | 云端 | 按分钟计费（更便宜） | 是 | 简单 | 注重成本的组织 |
| **Vonage** | 云端 | 按分钟计费 | 是 | 中等 | 国际覆盖 |
| **Plivo** | 云端 | 按分钟计费 | 是 | 中等 | 经济型云方案 |
| **Asterisk** | 自托管 | 仅 SIP 中继费用 | 是 (SIP.js) | 困难 | 最大隐私保护、大规模部署 |

## 价格对比

美国语音通话的大致每分钟费用（价格因地区和用量而异）：

| 提供商 | 呼入 | 呼出 | 电话号码 | 免费额度 |
|---|---|---|---|---|
| Twilio | $0.0085 | $0.014 | $1.15/月 | 试用额度 |
| SignalWire | $0.005 | $0.009 | $1.00/月 | 试用额度 |
| Vonage | $0.0049 | $0.0139 | $1.00/月 | 免费额度 |
| Plivo | $0.0055 | $0.010 | $0.80/月 | 试用额度 |
| Asterisk | SIP 中继费率 | SIP 中继费率 | 来自 SIP 提供商 | 不适用 |

所有云端提供商按分钟计费，精确到秒。Asterisk 的费用取决于您的 SIP 中继提供商和服务器托管成本。

## 功能支持矩阵

| 功能 | Twilio | SignalWire | Vonage | Plivo | Asterisk |
|---|---|---|---|---|---|
| 通话录音 | 是 | 是 | 是 | 是 | 是 |
| 实时语音转文字 | 是 | 是 | 是 | 是 | 是（通过桥接） |
| 语音验证码 | 是 | 是 | 是 | 是 | 是 |
| 语音信箱 | 是 | 是 | 是 | 是 | 是 |
| WebRTC 浏览器通话 | 是 | 是 | 是 | 是 | 是 (SIP.js) |
| Webhook 验证 | 是 | 是 | 是 | 是 | 自定义 (HMAC) |
| 并行振铃 | 是 | 是 | 是 | 是 | 是 |
| 排队/等待音乐 | 是 | 是 | 是 | 是 | 是 |

## 如何配置

1. 在管理侧边栏中导航到**设置**
2. 打开**电话服务提供商**部分
3. 从下拉菜单中选择您的提供商
4. 输入所需的凭证（每个提供商有不同的字段）
5. 以 E.164 格式输入您的热线电话号码（例如 `+15551234567`）
6. 点击**保存**
7. 在提供商的控制台中配置 Webhook，指向您的 Llamenos 实例

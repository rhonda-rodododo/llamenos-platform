作为管理员，您可以全面管理：志愿者、排班、通话设置、封禁列表和自定义字段。本指南涵盖关键的管理工作流程。

## 登录

使用在[设置](/docs/deploy)过程中生成的 `nsec`（Nostr 私钥）登录。登录页面接受 nsec 格式（`nsec1...`）。您的浏览器使用密钥对质询进行签名 — 私钥永远不会离开设备。

您也可以选择在设置中注册 WebAuthn 通行密钥，以便在其他设备上进行无密码登录。

## 管理志愿者

在侧边栏中导航到**志愿者**，可以：

- **添加志愿者** — 生成新的 Nostr 密钥对。将 nsec 安全地分享给志愿者（仅显示一次）。
- **创建邀请链接** — 生成一次性链接，志愿者可以使用该链接自行注册。
- **编辑** — 更新姓名、电话号码和角色。
- **移除** — 停用志愿者的访问权限。

志愿者的电话号码仅对管理员可见。当志愿者在班时，电话号码用于并行振铃。

## 配置排班

导航到**排班**以创建循环计划：

1. 点击**添加排班**
2. 设置名称，选择星期几，设置开始/结束时间
3. 使用可搜索的多选框分配志愿者
4. 保存 — 系统将自动将来电路由到当前排班的志愿者

在排班页面底部配置**备用组**。当没有排定的班次处于活动状态时，这些志愿者将被呼叫。

## 封禁列表

导航到**封禁**以管理被屏蔽的电话号码：

- **单条添加** — 输入 E.164 格式的电话号码（例如 +15551234567）
- **批量导入** — 粘贴多个号码，每行一个
- **移除** — 立即解除对某个号码的封禁

封禁即时生效。被封禁的来电者会听到拒绝消息并被断开连接。

## 通话设置

在**设置**中，您会找到以下几个部分：

### 垃圾电话防护

- **语音验证码** — 开启/关闭。开启后，来电者必须输入一个随机的 4 位数字代码。
- **速率限制** — 开启/关闭。在滑动时间窗口内限制每个电话号码的来电次数。

### 语音转文字

- **全局开关** — 为所有通话启用/禁用 Whisper 语音转文字。
- 各志愿者也可以通过自己的设置选择退出。

### 通话设置

- **排队超时** — 来电者在转到语音信箱前等待的时间（30-300 秒）。
- **语音信箱最大时长** — 录音的最大长度（30-300 秒）。

### 自定义备注字段

定义出现在备注表单中的结构化字段：

- 支持类型：文本、数字、选择（下拉菜单）、复选框、文本区域
- 配置验证：必填、最小/最大长度、最小/最大值
- 控制可见性：选择志愿者可以查看和编辑哪些字段
- 使用上/下箭头重新排列字段顺序
- 最多 20 个字段，每个选择字段最多 50 个选项

自定义字段的值与备注内容一起加密。服务器永远无法看到它们。

### 语音提示

为每种支持的语言录制自定义 IVR 音频提示。系统使用您的录音来播放问候语、验证码、排队和语音信箱流程。如果没有录音，则回退到文本转语音。

### WebAuthn 策略

可选择要求管理员、志愿者或两者都使用通行密钥。设为必需后，用户必须先注册通行密钥才能使用应用。

## 审计日志

**审计日志**页面显示按时间顺序排列的系统事件列表：登录、接听来电、创建备注、设置变更和管理操作。条目包含经哈希处理的 IP 地址和国家/地区元数据。使用分页浏览历史记录。

## 通话记录

**通话**页面显示所有通话及其状态、时长和志愿者分配情况。按日期范围筛选或按电话号码搜索。以符合 GDPR 的 JSON 格式导出数据。

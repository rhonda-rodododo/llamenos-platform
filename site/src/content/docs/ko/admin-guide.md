관리자로서 자원봉사자, 근무 일정, 통화 설정, 차단 목록, 사용자 정의 필드 등 모든 것을 관리합니다. 이 가이드는 주요 관리 워크플로를 다룹니다.

## 로그인

[설정](/docs/deploy) 과정에서 생성한 `nsec`(Nostr 비밀 키)로 로그인합니다. 로그인 페이지는 nsec 형식(`nsec1...`)을 허용합니다. 브라우저가 키로 챌린지에 서명합니다 — 비밀 키는 절대 기기를 벗어나지 않습니다.

선택적으로 설정에서 WebAuthn 패스키를 등록하여 추가 기기에서 비밀번호 없이 로그인할 수 있습니다.

## 자원봉사자 관리

사이드바에서 **자원봉사자**로 이동하여:

- **자원봉사자 추가** — 새로운 Nostr 키 쌍을 생성합니다. nsec를 안전하게 자원봉사자에게 공유하세요 (한 번만 표시됩니다).
- **초대 링크 생성** — 자원봉사자가 자가 등록할 수 있는 일회용 링크를 생성합니다.
- **편집** — 이름, 전화번호, 역할을 수정합니다.
- **제거** — 자원봉사자의 접근 권한을 비활성화합니다.

자원봉사자의 전화번호는 관리자만 볼 수 있습니다. 자원봉사자가 근무 중일 때 동시 벨울림에 사용됩니다.

## 근무 일정 구성

**근무 일정**으로 이동하여 반복 일정을 만듭니다:

1. **근무 추가** 클릭
2. 이름을 설정하고, 요일을 선택하고, 시작/종료 시간을 설정
3. 검색 가능한 다중 선택으로 자원봉사자 배정
4. 저장 — 시스템이 자동으로 활성 근무의 자원봉사자에게 전화를 라우팅합니다

근무 일정 페이지 하단에서 **대체 그룹**을 구성합니다. 예정된 근무가 활성화되지 않을 때 이 자원봉사자들에게 전화가 울립니다.

## 차단 목록

**차단**으로 이동하여 차단된 전화번호를 관리합니다:

- **단일 항목** — E.164 형식으로 전화번호를 입력 (예: +15551234567)
- **일괄 가져오기** — 여러 번호를 한 줄에 하나씩 붙여넣기
- **제거** — 즉시 차단 해제

차단은 즉시 적용됩니다. 차단된 발신자는 거절 메시지를 듣고 연결이 끊깁니다.

## 통화 설정

**설정**에서 여러 섹션을 찾을 수 있습니다:

### 스팸 방지

- **음성 CAPTCHA** — 켜기/끄기. 활성화하면 발신자가 무작위 4자리 숫자 코드를 입력해야 합니다.
- **속도 제한** — 켜기/끄기. 슬라이딩 시간 창 내에서 전화번호당 통화 횟수를 제한합니다.

### 음성 변환

- **전체 토글** — 모든 통화에 대해 Whisper 음성 변환을 활성화/비활성화합니다.
- 개별 자원봉사자도 자체 설정에서 옵트아웃할 수 있습니다.

### 통화 설정

- **대기열 시간 초과** — 발신자가 음성 사서함으로 이동하기 전 대기 시간 (30-300초).
- **음성 사서함 최대 시간** — 최대 녹음 길이 (30-300초).

### 사용자 정의 메모 필드

메모 양식에 표시되는 구조화된 필드를 정의합니다:

- 지원 유형: 텍스트, 숫자, 선택(드롭다운), 체크박스, 텍스트 영역
- 유효성 검사 구성: 필수, 최소/최대 길이, 최소/최대 값
- 가시성 제어: 자원봉사자가 볼 수 있고 편집할 수 있는 필드 선택
- 위/아래 화살표로 필드 순서 재배치
- 최대 20개 필드, 선택 필드당 최대 50개 옵션

사용자 정의 필드 값은 메모 내용과 함께 암호화됩니다. 서버는 절대 이를 볼 수 없습니다.

### 음성 안내

지원되는 각 언어에 대해 사용자 정의 IVR 오디오 안내를 녹음합니다. 시스템은 인사, CAPTCHA, 대기열, 음성 사서함 흐름에 녹음을 사용합니다. 녹음이 없는 경우 텍스트 음성 변환으로 대체됩니다.

### WebAuthn 정책

관리자, 자원봉사자 또는 모두에게 패스키를 요구하도록 선택할 수 있습니다. 필수로 설정하면 사용자는 앱을 사용하기 전에 패스키를 등록해야 합니다.

## 감사 로그

**감사 로그** 페이지는 시간순으로 시스템 이벤트를 보여줍니다: 로그인, 통화 응답, 메모 생성, 설정 변경, 관리자 작업. 항목에는 해시된 IP 주소와 국가 메타데이터가 포함됩니다. 페이지네이션을 사용하여 기록을 탐색합니다.

## 통화 기록

**통화** 페이지는 상태, 통화 시간, 자원봉사자 배정과 함께 모든 통화를 보여줍니다. 날짜 범위로 필터링하거나 전화번호로 검색합니다. GDPR 준수 JSON 형식으로 데이터를 내보냅니다.

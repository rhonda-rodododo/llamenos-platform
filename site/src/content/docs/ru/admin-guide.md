Как администратор, вы управляете всем: волонтёрами, сменами, настройками звонков, списками блокировки и пользовательскими полями. Это руководство описывает основные рабочие процессы администратора.

## Вход в систему

Войдите с помощью `nsec` (секретного ключа Nostr), сгенерированного при [настройке](/docs/deploy). Страница входа принимает формат nsec (`nsec1...`). Ваш браузер подписывает запрос с помощью ключа — секретный ключ никогда не покидает устройство.

По желанию зарегистрируйте ключ доступа WebAuthn в настройках для входа без пароля на дополнительных устройствах.

## Управление волонтёрами

Перейдите в раздел **Волонтёры** в боковой панели, чтобы:

- **Добавить волонтёра** — генерирует новую ключевую пару Nostr. Безопасно передайте nsec волонтёру (он показывается только один раз).
- **Создать ссылку-приглашение** — генерирует одноразовую ссылку, которую волонтёр может использовать для самостоятельной регистрации.
- **Редактировать** — обновить имя, номер телефона и роль.
- **Удалить** — деактивировать доступ волонтёра.

Номера телефонов волонтёров видны только администраторам. Они используются для параллельного вызова, когда волонтёр на смене.

## Настройка смен

Перейдите в раздел **Смены**, чтобы создать повторяющиеся расписания:

1. Нажмите **Добавить смену**
2. Задайте название, выберите дни недели и установите время начала/окончания
3. Назначьте волонтёров с помощью мультиселектора с поиском
4. Сохраните — система автоматически будет направлять звонки волонтёрам на активной смене

Настройте **Резервную группу** в нижней части страницы смен. Эти волонтёры будут получать вызовы, когда нет активной запланированной смены.

## Списки блокировки

Перейдите в раздел **Блокировки**, чтобы управлять заблокированными номерами:

- **Одиночная запись** — введите номер телефона в формате E.164 (например, +15551234567)
- **Массовый импорт** — вставьте несколько номеров, по одному на строку
- **Удалить** — мгновенно разблокировать номер

Блокировки вступают в силу немедленно. Заблокированные абоненты слышат сообщение об отказе и отключаются.

## Настройки звонков

В разделе **Настройки** вы найдёте несколько секций:

### Защита от спама

- **Голосовая CAPTCHA** — включить/выключить. При включении звонящие должны ввести случайный 4-значный код.
- **Ограничение частоты** — включить/выключить. Ограничивает количество звонков с одного номера в скользящем временном окне.

### Транскрипция

- **Глобальный переключатель** — включить/выключить транскрипцию Whisper для всех звонков.
- Отдельные волонтёры также могут отказаться через свои собственные настройки.

### Настройки звонков

- **Таймаут очереди** — как долго звонящие ждут перед переходом на голосовую почту (30-300 секунд).
- **Максимальная длительность голосовой почты** — максимальная длина записи (30-300 секунд).

### Пользовательские поля заметок

Определите структурированные поля, которые появляются в форме создания заметок:

- Поддерживаемые типы: текст, число, выпадающий список, флажок, текстовая область
- Настройка валидации: обязательное, мин./макс. длина, мин./макс. значение
- Управление видимостью: выберите, какие поля волонтёры могут видеть и редактировать
- Изменение порядка полей с помощью стрелок вверх/вниз
- Максимум 20 полей, максимум 50 вариантов в выпадающем списке

Значения пользовательских полей шифруются вместе с содержимым заметки. Сервер их никогда не видит.

### Голосовые подсказки

Запишите пользовательские аудиоподсказки IVR для каждого поддерживаемого языка. Система использует ваши записи для приветствия, CAPTCHA, очереди и голосовой почты. Где нет записи, используется синтез речи.

### Политика WebAuthn

По желанию потребуйте ключи доступа для администраторов, волонтёров или обоих. Когда это обязательно, пользователи должны зарегистрировать ключ доступа, прежде чем смогут использовать приложение.

## Журнал аудита

Страница **Журнал аудита** показывает хронологический список системных событий: входы в систему, ответы на звонки, создание заметок, изменения настроек и действия администратора. Записи включают хешированные IP-адреса и метаданные о стране. Используйте постраничную навигацию для просмотра истории.

## История звонков

Страница **Звонки** показывает все звонки со статусом, длительностью и назначенным волонтёром. Фильтруйте по диапазону дат или ищите по номеру телефона. Экспортируйте данные в GDPR-совместимом формате JSON.

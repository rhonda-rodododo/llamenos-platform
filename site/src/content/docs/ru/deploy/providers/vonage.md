Vonage (ранее Nexmo) предлагает широкое международное покрытие и конкурентоспособные цены. Он использует другую модель API по сравнению с Twilio — приложения Vonage объединяют ваш номер, вебхуки и учётные данные.

## Предварительные требования

- [Учётная запись Vonage](https://dashboard.nexmo.com/sign-up) (бесплатный кредит доступен)
- Ваш экземпляр Llamenos развёрнут и доступен по публичному URL

## 1. Создайте учётную запись Vonage

Зарегистрируйтесь в [панели управления Vonage API](https://dashboard.nexmo.com/sign-up). Подтвердите учётную запись и запишите **API Key** и **API Secret** с главной страницы панели управления.

## 2. Купите номер телефона

1. Перейдите в **Numbers** > **Buy numbers** в панели управления Vonage
2. Выберите страну и номер с возможностью **Voice**
3. Купите номер

## 3. Создайте приложение Vonage

Vonage группирует конфигурацию в «Приложения»:

1. Перейдите в **Applications** > **Create a new application**
2. Введите имя (например, «Llamenos Hotline»)
3. В разделе **Voice** включите его и установите:
   - **Answer URL**: `https://your-worker-url.com/telephony/incoming` (POST)
   - **Event URL**: `https://your-worker-url.com/telephony/status` (POST)
4. Нажмите **Generate new application**
5. Сохраните **Application ID**, показанный на странице подтверждения
6. Скачайте файл **закрытого ключа** — его содержимое понадобится для настройки

## 4. Привяжите номер телефона

1. Перейдите в **Numbers** > **Your numbers**
2. Нажмите на значок шестерёнки рядом с номером горячей линии
3. В разделе **Voice** выберите приложение, созданное на шаге 3
4. Нажмите **Save**

## 5. Настройте в Llamenos

1. Войдите как администратор
2. Перейдите в **Настройки** > **Провайдер телефонии**
3. Выберите **Vonage** из выпадающего списка
4. Введите:
   - **API Key**: с главной страницы панели управления Vonage
   - **API Secret**: с главной страницы панели управления Vonage
   - **Application ID**: из шага 3
   - **Phone Number**: купленный номер (формат E.164)
5. Нажмите **Сохранить**

## 6. Проверьте настройку

Позвоните на номер горячей линии. Вы должны услышать меню выбора языка. Убедитесь, что звонки направляются волонтёрам на смене.

## Настройка WebRTC (опционально)

Vonage WebRTC использует учётные данные приложения, которые вы уже создали:

1. В Llamenos перейдите в **Настройки** > **Провайдер телефонии**
2. Включите **Звонки через WebRTC**
3. Введите содержимое **закрытого ключа** (полный текст PEM из скачанного файла)
4. Нажмите **Сохранить**

Application ID уже настроен. Vonage генерирует RS256 JWT-токены с использованием закрытого ключа для аутентификации браузера.

## Особенности Vonage

- **NCCO vs TwiML**: Vonage использует NCCO (Nexmo Call Control Objects) в формате JSON вместо XML-разметки. Адаптер Llamenos автоматически генерирует правильный формат.
- **Формат Answer URL**: Vonage ожидает, что URL ответа вернёт JSON (NCCO), а не XML. Это обрабатывается адаптером.
- **Event URL**: Vonage отправляет события звонков (звонок, ответ, завершение) на URL событий в виде JSON POST-запросов.
- **Безопасность закрытого ключа**: Закрытый ключ хранится в зашифрованном виде. Он никогда не покидает сервер — используется только для генерации краткосрочных JWT-токенов.

## Устранение неполадок

- **«Application not found»**: Убедитесь, что Application ID совпадает точно. Найти его можно в разделе **Applications** панели управления Vonage.
- **Нет входящих звонков**: Убедитесь, что номер телефона привязан к правильному приложению (шаг 4).
- **Ошибки закрытого ключа**: Вставьте полное содержимое PEM, включая строки `-----BEGIN PRIVATE KEY-----` и `-----END PRIVATE KEY-----`.
- **Формат международных номеров**: Vonage требует формат E.164. Включите `+` и код страны.

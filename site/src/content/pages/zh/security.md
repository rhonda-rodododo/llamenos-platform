## 如果您的托管提供商收到传票

| 他们可以提供 | 他们无法提供 |
|-------------|-------------|
| 通话/消息元数据（时间、时长） | 备注内容、转录文本、报告正文 |
| 加密的数据库数据块 | 解密密钥（存储在您的设备上） |
| 哪些志愿者在什么时候活跃 | 每条备注的加密密钥（临时密钥） |
| SMS/WhatsApp 消息内容 | 您的 HMAC 密钥（用于反转电话号码哈希） |

**服务器存储它无法读取的数据。** 元数据（何时、多长时间、谁）是可见的。内容（说了什么、写了什么）不可见。

## 按功能

您的隐私暴露取决于您启用了哪些频道：

### 语音通话

| 如果您使用... | 第三方可以访问 | 服务器可以访问 | E2EE 内容 |
|--------------|--------------|--------------|-----------|
| Twilio/SignalWire/Vonage/Plivo | 通话音频（实时）、通话记录 | 通话元数据 | 备注、转录文本 |
| 自托管 Asterisk | 无（您完全控制） | 通话元数据 | 备注、转录文本 |
| 浏览器对浏览器 (WebRTC) | 无 | 通话元数据 | 备注、转录文本 |

**电话服务提供商传票**：他们有通话详细记录（时间、电话号码、时长）。他们没有通话备注或转录文本。录音默认是禁用的。

**转录窗口**：在约 30 秒的转录过程中，音频由 Cloudflare Workers AI 处理。转录后，仅存储加密文本。

### 文字消息

| 频道 | 提供商访问 | 服务器存储 | 备注 |
|------|-----------|-----------|------|
| SMS | 您的电话服务提供商可以读取所有消息 | 明文 | SMS 的固有限制 |
| WhatsApp | Meta 可以读取所有消息 | 明文 | WhatsApp Business API 要求 |
| Signal | Signal 网络是 E2EE 的，但 signal-cli 桥接会解密 | 明文 | 比 SMS 好，但不是零知识 |

**消息提供商传票**：SMS 提供商拥有完整的消息内容。Meta 拥有 WhatsApp 内容。Signal 消息到桥接器为止是 E2EE 的，但桥接器（运行在您的服务器上）有明文。

**未来改进**：我们正在探索 E2EE 消息存储，服务器仅存储密文。参见[计划中的功能](#计划中的功能)。

### 备注、转录文本和报告

所有志愿者撰写的内容都是端到端加密的：

- 每条备注使用唯一的随机密钥（前向保密）
- 密钥为志愿者和管理员分别封装
- 服务器仅存储密文
- 解密在浏览器中进行

**设备扣押**：没有您的 PIN，攻击者得到的是一个加密数据块。使用 600K 次 PBKDF2 迭代的 6 位 PIN 在 GPU 硬件上需要数小时进行暴力破解。

## 志愿者电话号码隐私

当志愿者在个人手机上接听来电时，他们的号码会暴露给电话服务提供商。

| 场景 | 电话号码对谁可见 |
|------|----------------|
| PSTN 通话到志愿者手机 | 电话服务提供商、手机运营商 |
| 浏览器对浏览器 (WebRTC) | 无人（音频留在浏览器中） |
| 自托管 Asterisk + SIP 电话 | 仅您的 Asterisk 服务器 |

**保护志愿者电话号码**：使用基于浏览器的通话 (WebRTC) 或提供连接到自托管 Asterisk 的 SIP 电话。

**未来改进**：原生桌面和移动应用用于接听来电，无需暴露个人电话号码。

## 计划中的功能

我们正在进行改进以减少信任要求：

| 功能 | 状态 | 隐私益处 |
|------|------|---------|
| E2EE 消息存储 | 计划中 | SMS/WhatsApp/Signal 以密文存储 |
| 客户端语音转文字 | 计划中 | 音频永远不离开浏览器 |
| 原生来电接听应用 | 计划中 | 不暴露个人电话号码 |
| 可重现构建 | 计划中 | 验证部署代码与源代码一致 |
| 自托管 Signal 桥接 | 可用 | 在您自己的基础设施上运行 signal-cli |

## 总结表

| 数据类型 | 是否加密 | 对服务器可见 | 传票下可获取 |
|---------|---------|-------------|-------------|
| 通话备注 | 是 (E2EE) | 否 | 仅密文 |
| 转录文本 | 是 (E2EE) | 否 | 仅密文 |
| 报告 | 是 (E2EE) | 否 | 仅密文 |
| 文件附件 | 是 (E2EE) | 否 | 仅密文 |
| 通话元数据 | 否 | 是 | 是 |
| 志愿者身份 | 静态加密 | 仅管理员 | 是（需要一定努力） |
| 来电者电话号码哈希 | HMAC 哈希 | 仅哈希 | 哈希（没有您的密钥不可反转） |
| SMS 内容 | 否 | 是 | 是 |
| WhatsApp 内容 | 否 | 是 | 是（也可从 Meta 获取） |
| Signal 内容 | 否 | 是 | 是（从您的服务器） |

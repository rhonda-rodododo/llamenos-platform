## 如果您的托管服务商收到传票

| 他们**能**提供的 | 他们**无法**提供的 |
|------------------|---------------------|
| 通话/消息元数据（时间、时长） | 笔记内容、转录文本、报告正文 |
| 加密的数据库二进制对象 | 志愿者姓名（端到端加密） |
| 哪些志愿者账号在何时活跃 | 联系人目录记录（端到端加密） |
| | 消息内容（到达时即加密，以密文存储） |
| | 解密密钥（受您的 PIN、身份提供商账号及可选的硬件安全密钥保护） |
| | 单条笔记的加密密钥（临时密钥——封装后即销毁） |
| | 您的 HMAC 密钥（用于反向解析电话号码哈希） |

**服务器存储的是它自己无法读取的数据。** 元数据（何时、多长、哪些账号）可见。内容（说了什么、写了什么、您的联系人是谁）不可见。

## 按功能分类

您的隐私暴露程度取决于您启用了哪些通道：

### 语音通话

| 使用方式 | 第三方可访问 | 服务器可访问 | 端到端加密的内容 |
|----------|-------------|-------------|-----------------|
| Twilio/SignalWire/Vonage/Plivo | 通话音频（实时）、通话记录 | 通话元数据 | 笔记、转录文本 |
| 自托管 Asterisk | 无（由您完全控制） | 通话元数据 | 笔记、转录文本 |
| 浏览器对浏览器（WebRTC） | 无 | 通话元数据 | 笔记、转录文本 |

**电话服务商传票**：他们拥有通话详情记录（时间、电话号码、时长）。他们**没有**通话笔记或转录文本。录音功能默认关闭。

**转录**：转录完全在您的浏览器中通过设备端 AI 完成。**音频永远不会离开您的设备。** 只有加密后的转录文本会被存储。

### 短信与消息

| 渠道 | 服务商可访问的内容 | 服务器存储方式 | 备注 |
|------|-------------------|---------------|------|
| 短信（SMS） | 您的电话服务商可读取所有消息 | **加密存储** | 服务商保留原始消息 |
| WhatsApp | Meta 可读取所有消息 | **加密存储** | 服务商保留原始消息 |
| Signal | Signal 网络端到端加密，但桥接器在到达时解密 | **加密存储** | 优于短信，但非零知识 |

**消息在到达您的服务器时即被加密。** 服务器仅存储密文。您的电话或消息服务商可能仍持有原始消息——这是那些平台的局限性，不是我们能改变的。

**消息服务商传票**：短信服务商拥有完整消息内容。Meta 拥有 WhatsApp 内容。Signal 消息在到达桥接器前是端到端加密的，但桥接器（运行在您的服务器上）在重新加密存储前会解密。在所有情况下，**您的服务器上只有密文**——托管服务商无法读取消息内容。

### 笔记、转录文本和报告

所有志愿者撰写的内容均端到端加密：

- 每条笔记使用**唯一的随机密钥**（前向保密——泄露一条笔记不会影响其他笔记）
- 密钥分别为志愿者和每位管理员单独封装
- 服务器仅存储密文
- 解密在浏览器中完成
- **自定义字段、报告内容和附件均单独加密**

**设备被没收时**：没有您的 PIN **和**身份提供商账号访问权限，攻击者得到的只是一个计算上不可行破解的加密数据块。如果您还使用了硬件安全密钥，则有**三个独立因素**保护您的数据。

## 志愿者电话号码隐私

当志愿者在个人手机上接听电话时，其号码会暴露给您的电话服务商。

| 场景 | 电话号码对谁可见 |
|------|-----------------|
| PSTN 呼叫到志愿者手机 | 电话服务商、手机运营商 |
| 浏览器对浏览器（WebRTC） | 无人可见（音频留在浏览器中） |
| 自托管 Asterisk + SIP 话机 | 仅您的 Asterisk 服务器 |

**保护志愿者电话号码的方法**：使用浏览器通话（WebRTC）或提供连接到自托管 Asterisk 的 SIP 话机。

## 近期已上线

以下改进现已可用：

| 功能 | 隐私收益 |
|------|---------|
| 加密消息存储 | SMS、WhatsApp 和 Signal 消息以密文形式存储在您的服务器上 |
| 设备端转录 | 音频永远不会离开您的浏览器——完全在设备上处理 |
| 多因素密钥保护 | 您的加密密钥受 PIN、身份提供商和可选硬件安全密钥保护 |
| 硬件安全密钥 | 物理密钥增加第三因素，无法被远程攻破 |
| 可复现构建 | 验证部署的代码与公开源码一致 |
| 加密联系人目录 | 联系人记录、关系和备注均端到端加密 |

## 仍在计划中

| 功能 | 隐私收益 |
|------|---------|
| 原生来电接听应用 | 不再暴露个人电话号码 |

## 汇总表

| 数据类型 | 是否加密 | 服务器可见 | 可依传票获取 |
|----------|---------|-----------|-------------|
| 通话笔记 | 是（端到端） | 否 | 仅密文 |
| 转录文本 | 是（端到端） | 否 | 仅密文 |
| 报告 | 是（端到端） | 否 | 仅密文 |
| 文件附件 | 是（端到端） | 否 | 仅密文 |
| 联系人记录 | 是（端到端） | 否 | 仅密文 |
| 志愿者身份 | 是（端到端） | 否 | 仅密文 |
| 团队/角色元数据 | 是（加密） | 否 | 仅密文 |
| 自定义字段定义 | 是（加密） | 否 | 仅密文 |
| SMS/WhatsApp/Signal 内容 | 是（在您的服务器上） | 否 | 您的服务器上为密文；服务商可能持有原文 |
| 通话元数据 | 否 | 是 | 是 |
| 来电者电话哈希 | HMAC 哈希 | 仅哈希值 | 哈希值（没有您的密钥无法反向解析） |

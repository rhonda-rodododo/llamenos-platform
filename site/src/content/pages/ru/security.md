## Если ваш хостинг-провайдер получает повестку

| Они МОГУТ предоставить | Они НЕ МОГУТ предоставить |
|------------------|---------------------|
| Метаданные звонков/сообщений (время, продолжительность) | Содержимое заметок, транскрипты, тела отчётов |
| Зашифрованные блобы базы данных | Ключи дешифрования (хранятся на ваших устройствах) |
| Каких волонтёров и когда активировали | Ключи шифрования для каждой заметки (эфемерные) |
| Содержимое SMS/WhatsApp-сообщений | Ваш HMAC-секрет для реверсии хешей телефонов |

**Сервер хранит данные, которые не может прочитать.** Метаданные (когда, как долго, кто) видны. Содержимое (что было сказано, что было написано) — нет.

## По функциям

Ваша уязвимость конфиденциальности зависит от того, какие каналы вы включаете:

### Голосовые звонки

| Если вы используете... | Третьи стороны могут получить доступ | Сервер может получить доступ | Контент E2EE |
|---------------|-------------------------|-------------------|--------------|
| Twilio/SignalWire/Vonage/Plivo | Аудио звонка (в реальном времени), записи звонков | Метаданные звонков | Заметки, транскрипты |
| Самостоятельный Asterisk | Ничего (вы контролируете) | Метаданные звонков | Заметки, транскрипты |
| Браузер-браузер (WebRTC) | Ничего | Метаданные звонков | Заметки, транскрипты |

**Повестка провайдера телефонии**: они имеют детальные записи звонков (время, номера телефонов, продолжительность). У них НЕТ заметок звонков или транскриптов. Запись по умолчанию отключена.

**Окно транскрипции**: В течение ~30 секунд транскрипции аудио обрабатывается Cloudflare Workers AI. После транскрипции хранится только зашифрованный текст.

### Текстовые сообщения

| Канал | Доступ провайдера | Хранение на сервере | Примечания |
|---------|-----------------|----------------|-------|
| SMS | Ваш провайдер телефонии читает все сообщения | Открытый текст | Неотъемлемое ограничение SMS |
| WhatsApp | Meta читает все сообщения | Открытый текст | Требование WhatsApp Business API |
| Signal | Сеть Signal E2EE, но мост signal-cli расшифровывает | Открытый текст | Лучше SMS, не без нулевых знаний |

**Повестка провайдера обмена сообщениями**: провайдер SMS имеет полное содержимое сообщений. Meta имеет содержимое WhatsApp. Сообщения Signal E2EE до моста, но мост (работающий на вашем сервере) имеет открытый текст.

**Будущее улучшение**: Мы изучаем хранение сообщений с E2EE, где сервер хранит только шифртекст. См. [дорожную карту](#что-планируется).

### Заметки, транскрипты и отчёты

Весь контент, написанный волонтёрами, зашифрован сквозным шифрованием:

- Каждая заметка использует уникальный случайный ключ (прямая секретность)
- Ключи оборачиваются отдельно для волонтёра и администратора
- Сервер хранит только шифртекст
- Дешифрование происходит в браузере

**Изъятие устройства**: Без PIN атакующие получают зашифрованный блоб. 6-значный PIN с 600К итерациями PBKDF2 занимает часы при брутфорсе на GPU-оборудовании.

## Конфиденциальность номеров телефонов волонтёров

Когда волонтёры получают звонки на личные телефоны, их номера открыты для провайдера телефонии.

| Сценарий | Номер телефона виден для |
|----------|------------------------|
| PSTN-звонок на телефон волонтёра | Провайдер телефонии, телефонный оператор |
| Браузер-браузер (WebRTC) | Никто (аудио остаётся в браузере) |
| Самостоятельный Asterisk + SIP-телефон | Только ваш Asterisk-сервер |

**Для защиты номеров телефонов волонтёров**: используйте браузерные звонки (WebRTC) или обеспечьте SIP-телефоны, подключённые к самостоятельному Asterisk.

**Будущее улучшение**: Нативные настольные и мобильные приложения для получения звонков без раскрытия личных номеров телефонов.

## Что планируется

Мы работаем над улучшениями для снижения требований к доверию:

| Функция | Статус | Преимущество для конфиденциальности |
|---------|--------|-----------------|
| Хранение сообщений с E2EE | Запланировано | SMS/WhatsApp/Signal хранятся как шифртекст |
| Транскрипция на стороне клиента | Запланировано | Аудио никогда не покидает браузер |
| Нативные приложения для получения звонков | Запланировано | Личные номера телефонов не раскрываются |
| Воспроизводимые сборки | Запланировано | Проверяйте соответствие развёрнутого кода исходному |
| Самостоятельный мост Signal | Доступно | Запускайте signal-cli на своей инфраструктуре |

## Сводная таблица

| Тип данных | Зашифровано | Видно серверу | Получаемо по повестке |
|-----------|-----------|-------------------|---------------------------|
| Заметки звонков | Да (E2EE) | Нет | Только шифртекст |
| Транскрипты | Да (E2EE) | Нет | Только шифртекст |
| Отчёты | Да (E2EE) | Нет | Только шифртекст |
| Файловые вложения | Да (E2EE) | Нет | Только шифртекст |
| Метаданные звонков | Нет | Да | Да |
| Идентификаторы волонтёров | Зашифровано в покое | Только администраторы | Да (с усилиями) |
| Хеши телефонов звонящих | HMAC-хеш | Только хеш | Хеш (не обратимый без вашего секрета) |
| Содержимое SMS | Нет | Да | Да |
| Содержимое WhatsApp | Нет | Да | Да (также от Meta) |
| Содержимое Signal | Нет | Да | Да (с вашего сервера) |

## Многопровайдерная телефония

**5 голосовых провайдеров** — выбирайте из Twilio, SignalWire, Vonage, Plivo или самостоятельно размещённого Asterisk. Настраивайте провайдера в интерфейсе настроек администратора или в мастере настройки. Меняйте провайдеров в любое время без изменения кода.

**Вызовы WebRTC через браузер** — волонтёры могут отвечать на звонки прямо в браузере без телефона. Генерация WebRTC-токенов для Twilio, SignalWire, Vonage и Plivo. Настраиваемые предпочтения звонков для каждого волонтёра (телефон, браузер или оба).

## Маршрутизация звонков

**Параллельный звонок** — когда звонит абонент, одновременно звонят все волонтёры, находящиеся на смене и не занятые. Первый волонтёр, ответивший на звонок, получает его; остальные звонки немедленно прекращаются.

**Расписание смен** — создавайте повторяющиеся смены с конкретными днями и временными диапазонами. Назначайте волонтёров на смены. Система автоматически маршрутизирует звонки к дежурным.

**Очередь с музыкой ожидания** — если все волонтёры заняты, звонящие попадают в очередь с настраиваемой музыкой ожидания. Тайм-аут очереди настраивается (30–300 секунд). Если никто не отвечает, звонки переводятся на голосовую почту.

**Резервная голосовая почта** — звонящие могут оставить голосовое сообщение (до 5 минут), если ни один волонтёр не отвечает. Голосовые сообщения транскрибируются через Whisper AI и шифруются для проверки администратором.

## Зашифрованные заметки

**Заметки со сквозным шифрованием** — волонтёры пишут заметки во время и после звонков. Заметки шифруются на стороне клиента с использованием ECIES (secp256k1 + XChaCha20-Poly1305) перед отправкой из браузера. Сервер хранит только шифртекст.

**Двойное шифрование** — каждая заметка шифруется дважды: один раз для волонтёра, написавшего её, и один раз для администратора. Оба могут расшифровывать независимо. Никто другой не может прочитать содержимое.

**Пользовательские поля** — администраторы определяют пользовательские поля для заметок: текст, число, выбор, флажок, текстовая область. Поля шифруются вместе с содержимым заметки.

**Автосохранение черновиков** — заметки автоматически сохраняются как зашифрованные черновики в браузере. Если страница перезагружается или волонтёр переходит на другую страницу, его работа сохраняется. Черновики удаляются при выходе из системы.

## Транскрипция на основе ИИ

**Транскрипция на базе Whisper** — записи звонков транскрибируются с помощью Cloudflare Workers AI с моделью Whisper. Транскрипция происходит на стороне сервера, затем транскрипт шифруется перед хранением.

**Элементы управления** — администратор может включать/выключать транскрипцию глобально. Волонтёры могут отказаться индивидуально. Оба переключателя независимы.

**Зашифрованные транскрипты** — транскрипты используют то же ECIES-шифрование, что и заметки. Сохранённый транскрипт — только шифртекст.

## Защита от спама

**Голосовая CAPTCHA** — опциональное обнаружение голосовых ботов: звонящие слышат случайное 4-значное число и должны ввести его на клавиатуре. Блокирует автоматический набор, оставаясь доступным для реальных звонящих.

**Ограничение скорости** — ограничение скорости со скользящим окном по номеру телефона, сохранённое в хранилище Durable Object. Выживает при перезапуске Worker. Настраиваемые пороговые значения.

**Списки блокировки в реальном времени** — администраторы управляют списками блокировки номеров телефонов с одиночным вводом или массовым импортом. Блокировки вступают в силу немедленно. Заблокированные звонящие слышат сообщение об отказе.

**Пользовательские IVR-подсказки** — записывайте пользовательские голосовые подсказки для каждого поддерживаемого языка. Система использует ваши записи для IVR-сценариев, возвращаясь к синтезу речи при отсутствии записи.

## Многоканальный обмен сообщениями

**SMS** — двусторонний SMS через Twilio, SignalWire, Vonage или Plivo. Автоответ с настраиваемыми приветственными сообщениями. Сообщения поступают в представление переписки с треддингом.

**WhatsApp Business** — подключение через Meta Cloud API (Graph API v21.0). Поддержка шаблонных сообщений для инициирования переговоров в рамках 24-часового окна. Поддержка медиасообщений для изображений, документов и аудио.

**Signal** — конфиденциальный обмен сообщениями через самостоятельно размещённый мост signal-cli-rest-api. Мониторинг работоспособности с плавным снижением функциональности. Транскрипция голосовых сообщений через Workers AI Whisper.

**Треддинговые переговоры** — все каналы обмена сообщениями объединяются в единое представление переговоров. Пузыри сообщений с временными метками и индикаторами направления. Обновления в реальном времени через WebSocket.

## Зашифрованные отчёты

**Роль репортёра** — специальная роль для людей, отправляющих подсказки или отчёты. Репортёры видят упрощённый интерфейс только с отчётами и помощью. Приглашаются через тот же процесс, что и волонтёры, с выбором роли.

**Зашифрованные отправки** — содержимое тела отчёта шифруется с использованием ECIES перед отправкой из браузера. Открытые заголовки для сортировки, зашифрованное содержимое для конфиденциальности. Файловые вложения шифруются отдельно.

**Рабочий процесс отчётов** — категории для организации отчётов. Отслеживание статуса (открыт, принят, решён). Администраторы могут принимать отчёты и отвечать с помощью треддинговых зашифрованных ответов.

## Панель администратора

**Мастер настройки** — пошаговая настройка при первом входе администратора. Выберите, какие каналы включить (Голос, SMS, WhatsApp, Signal, Отчёты), настройте провайдеров и задайте название горячей линии.

**Контрольный список начала работы** — виджет панели управления, отслеживающий ход настройки: настройка канала, подключение волонтёров, создание смены.

**Мониторинг в реальном времени** — просматривайте активные звонки, звонящих в очереди, переговоры и статус волонтёров в реальном времени через WebSocket. Метрики обновляются мгновенно.

**Управление волонтёрами** — добавляйте волонтёров со сгенерированными парами ключей, управляйте ролями (волонтёр, администратор, репортёр), просматривайте онлайн-статус. Ссылки-приглашения для самостоятельной регистрации с выбором роли.

**Журналирование аудита** — каждый принятый звонок, созданная заметка, отправленное сообщение, поданный отчёт, изменённая настройка и действие администратора записывается. Просмотрщик с пагинацией для администраторов.

**История звонков** — история звонков с поиском и фильтрацией по диапазонам дат, поиску по номеру телефона и назначению волонтёра. Экспорт данных в соответствии с GDPR.

**Встроенная помощь** — разделы часто задаваемых вопросов, руководства по ролям, краткие справочные карточки по горячим клавишам и безопасности. Доступны из боковой панели и командной палитры.

## Опыт волонтёра

**Командная палитра** — нажмите Ctrl+K (или Cmd+K на Mac) для мгновенного доступа к навигации, поиску, быстрому созданию заметок и смене темы. Команды только для администратора фильтруются по роли.

**Уведомления в реальном времени** — входящие звонки запускают звонок в браузере, push-уведомление и мигающий заголовок вкладки. Каждый тип уведомлений можно независимо переключать в настройках.

**Присутствие волонтёров** — администраторы видят количество онлайн, офлайн и на перерыве в реальном времени. Волонтёры могут переключать статус перерыва в боковой панели, чтобы приостановить входящие звонки, не покидая смену.

**Горячие клавиши** — нажмите ? для просмотра всех доступных сочетаний клавиш. Навигируйте по страницам, открывайте командную палитру и выполняйте общие действия без использования мыши.

**Автосохранение черновиков** — заметки автоматически сохраняются как зашифрованные черновики в браузере. Если страница перезагружается или волонтёр переходит на другую страницу, его работа сохраняется. Черновики удаляются из localStorage при выходе из системы.

**Зашифрованный экспорт данных** — экспортируйте заметки в виде зашифрованного файла в соответствии с GDPR (.enc) с использованием собственного ключа волонтёра. Только оригинальный автор может расшифровать экспорт.

**Тёмная/светлая темы** — переключайтесь между тёмным режимом, светлым режимом или следуйте системной теме. Предпочтение сохраняется для сессии.

## Многоязычность и мобильность

**12+ языков** — полные переводы интерфейса: английский, испанский, китайский, тагальский, вьетнамский, арабский, французский, гаитянский креольский, корейский, русский, хинди, португальский и немецкий. Поддержка RTL для арабского.

**Прогрессивное веб-приложение** — устанавливается на любое устройство через браузер. Service worker кэширует оболочку приложения для офлайн-запуска. Push-уведомления о входящих звонках.

**Дизайн, ориентированный на мобильные устройства** — адаптивный макет для телефонов и планшетов. Складная боковая панель, удобные для сенсорного управления элементы и адаптивные компоновки.

## Аутентификация и управление ключами

**Хранилище ключей, защищённое PIN** — ваш секретный ключ шифруется с 6-значным PIN с использованием PBKDF2 (600 000 итераций) + XChaCha20-Poly1305. Необработанный ключ никогда не попадает в sessionStorage или другой API браузера — он хранится только в замыкании в памяти, обнуляемом при блокировке.

**Автоблокировка** — менеджер ключей блокируется автоматически после тайм-аута простоя или когда вкладка браузера скрыта. Введите PIN для разблокировки. Настраиваемая длительность простоя.

**Привязка устройств** — настраивайте новые устройства без раскрытия секретного ключа. Отсканируйте QR-код или введите короткий код подготовки. Использует эфемерный обмен ключами ECDH для безопасной передачи зашифрованных ключей между устройствами. Комнаты подготовки истекают через 5 минут.

**Ключи восстановления** — при первоначальной настройке вы получаете ключ восстановления в формате Base32 (128 бит энтропии). Это заменяет старый процесс отображения nsec. Обязательная загрузка зашифрованной резервной копии перед продолжением.

**Прямая секретность для каждой заметки** — каждая заметка шифруется уникальным случайным ключом, затем этот ключ оборачивается через ECIES для каждого авторизованного читателя. Компрометация ключа идентификатора не раскрывает прошлые заметки.

**Аутентификация через пары ключей Nostr** — волонтёры аутентифицируются с парами ключей, совместимыми с Nostr (nsec/npub). Верификация подписи BIP-340 Schnorr. Без паролей, без адресов электронной почты.

**Пасскеи WebAuthn** — опциональная поддержка пасскеев для входа на нескольких устройствах. Зарегистрируйте аппаратный ключ или биометрию, затем входите без ввода секретного ключа.

**Управление сессиями** — двухуровневая модель доступа: "аутентифицирован, но заблокирован" (только токен сессии) против "аутентифицирован и разблокирован" (введён PIN, полный доступ к криптографии). 8-часовые токены сессий с предупреждениями о тайм-ауте простоя.
